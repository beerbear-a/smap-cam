# ZootoCam 現状仕様

## コンセプト

フィルム体験を核にしたカメラアプリ。ロール単位で撮影し、現像して振り返る体験を現代的なUIで提供する。
撮影メモは自動入力され、場所チェックインと結び付けて記録する（観光地API連携は後工程）。

## 体験フロー

1) 場所チェックインでロールを作成  
2) 撮影（フィルム/インスタント）  
3) 現像 → アルバムで振り返り  
4) 共有（透かし合成、コンタクトシート）

## 主要機能

- カメラ
  - フィルム/インスタント撮影
  - フィルムビュー/通常ビュー切替
  - フィルムルック選択（FILM）
  - グリッド、タイマー、ライトリーク
- 記録
  - 写真メモの自動入力（iOS Vision）
  - ロール単位のメモ/テーマ
- 現像/アルバム
  - ロール現像
  - インデックスシート生成
  - アルバム閲覧
- 共有
  - 透かし合成
  - コンタクトシート出力（square/index/story）
- 図鑑
  - 出会い済み/未発見タブ
  - コンプリート率

## 場所チェックイン

- 現在は内部データで候補を表示
- 将来は観光地API（Google Places / Mapbox Places）へ切替予定

## デバッグ

- 設定にデバッグモード
- 場所機能のON/OFF
- フィルムシェーダー切替
