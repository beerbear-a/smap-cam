## 5. マーケティング戦略

### ゼロコスト獲得（オーガニック）

```
透かしによる自然拡散（最重要）
└── 投稿写真 → 「これ何のアプリ？」 → DL

Twitter/X コミュニティ活用
└── #動物園 #動物写真 のヘビーユーザーへの種まき
    ターゲットアカウント例:
    ├── フォロワー1,000〜10,000の動物写真垢
    └── 動物園の公式アカウント

note との親和性
└── 「ZOOSMAPで撮った写真を使った観察日記」
    を note でコンテンツ化 → 流入
```

### KOL戦略（Key Opinion Leader）
```
予算: ¥0（初期）

対象: 動物園写真を定期投稿しているTwitter/Instagramユーザー
アプローチ: Proコードをプレゼント + フィードバック依頼
期待効果: 初期ユーザー50〜200人獲得
```

## 6. リスクと対策

| リスク | 対策 |
|--------|------|
| ユーザーが集まらない | 透かし拡散 + KOL戦略。52本で黒字なので最悪ケースでも損失なし |
| DBメンテが大変 | ユーザー報告機能 + コミュニティ運営 |
| 大企業が参入 | 動物愛好家コミュニティとの絆・先行優位 |
