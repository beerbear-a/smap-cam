## 2. 競合分析

### 直接競合

| アプリ | 強み | 弱み | ZOOSMAPとの差異 |
|--------|------|------|----------------|
| **iNaturalist** | 世界最大の生物観察DB・AI同定精度高 | 英語中心・科学者向け・カメラ機能なし | 日本語・エンタメ寄り・カメラが主役 |
| **Seek** | 子ども向け・ゲーミフィケーション | 動物園特化なし・記録が弱い | 大人向け・SNS発信に特化 |
| **eBird** | 鳥類特化・科学的精度 | 鳥のみ・発信機能なし | 哺乳類も対象・SNS連携 |

### 間接競合（カメラアプリ）

| アプリ | 強み | 弱み | ZOOSMAPとの差異 |
|--------|------|------|----------------|
| **Dazz Camera** | フィルム質感・人気 | 記録機能なし・動物特化なし | 記録+図鑑が核心 |
| **VSCO** | 高品質フィルター・コミュニティ | 高価・一般向け | 動物特化・安価 |
| **Halide** | プロ向け高機能 | 記録なし・高価 | より親しみやすい |

### 間接競合（ライフログ）

| アプリ | 強み | 弱み | ZOOSMAPとの差異 |
|--------|------|------|----------------|
| **Notion** | 柔軟な記録 | カメラ機能なし・煩雑 | ワンタップで記録完了 |
| **Google Photos** | 自動整理・検索 | 動物特化でない | 図鑑・コレクション体験 |

### 結論: ブルーオーシャン
```
カメラ品質 × 動物園特化記録 × SNS発信補助
この3つを組み合わせたアプリは存在しない
```

## 5. エコシステム展開可能性

ZOOSMAPのコアエンジン（カメラ + チェックイン + 記録 + 発信）は他ドメインに転用可能:

```
ZOOSMAP       → 動物園・野生動物  ← まずここで勝つ
FestivalCAM   → 夏フェス・ライブ
TravelCAM     → 旅行・世界遺産
SportCAM      → スポーツ観戦
BotanicalCAM  → 植物園・花
```

**各アプリが透かしでお互いを宣伝し合うエコシステム。**

## 6. リスク評価

| リスク | 確率 | 影響 | 対策 |
|--------|------|------|------|
| ニッチすぎてユーザーが集まらない | 中 | 高 | 透かしでのオーガニック拡散に賭ける |
| iNaturalistが日本語強化・動物園特化 | 低 | 高 | スピードで先行・コミュニティ形成 |
| 動物DBのメンテが追いつかない | 中 | 中 | ユーザー報告機能で分散化 |
| カメラアプリとの差別化不足 | 低 | 中 | 図鑑・記録機能が核心・カメラは手段 |
