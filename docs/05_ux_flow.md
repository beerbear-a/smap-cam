# ZOOSMAP - UXフロー設計

> バージョン: 1.0
> 最終更新: 2026-03-13

---

## 1. 画面一覧

| 画面 | 役割 | 起動条件 |
|------|------|---------|
| MapScreen | ホーム。出会いの地図 | 常時 |
| CheckinScreen | 動物園選択 | 撮影前 |
| CameraScreen | 撮影 | チェックイン後 |
| QuickTagScreen | 動物タグ付け | 撮影直後 |
| ZukanScreen | 図鑑一覧 | タブ切替 |
| AnimalDetailScreen | 動物の詳細 + 自分の記録 | 図鑑タップ |
| EncounterDetailScreen | 特定の出会いの詳細 | 記録タップ |
| ExportScreen | 透かし + シェア | 任意 |
| PurchaseScreen | 買い切り（¥370） | 有料機能タップ時 |
| SettingsScreen | ユーザー名・設定 | 設定タブ |

---

## 4. 各画面の詳細設計

### CameraScreen

```
┌─────────────────────────────────┐
│  上野動物園  ●撮影中       ✕  │  ← ヘッダー
├─────────────────────────────────┤
│                                 │
│                                 │
│       [カメラプレビュー]         │
│                                 │
│                                 │
├─────────────────────────────────┤
│  ◁ Natural │ Kodak │ Fuji ▷   │  ← LUT スワイプ
├────────┬────────────────┬───────┤
│  📁   │      ⚪         │  ⚡  │
│ 読込み │   SHUTTER      │フラッシュ│
└────────┴────────────────┴───────┘
```

**インタラクション:**
- LUTはスワイプで切替（リアルタイムプレビュー）
- シャッターボタンは長押し不可（フィルムカメラ的）
- ✕ で MapScreen に戻る

---

### QuickTagScreen

```
┌─────────────────────────────────┐
│                                 │
│     [撮った写真サムネイル]       │
│                                 │
├─────────────────────────────────┤
│  何の動物？                     │
│  ┌─────────────────────────┐   │
│  │ 🔍 レッサーパンダ        │   │
│  └─────────────────────────┘   │
│                                 │
│  よく撮る動物:                  │
│  [🐼レッサーパンダ] [🦁ライオン] │  ← 履歴から
│                                 │
│  [ 一言メモ（任意）         ]   │
│                                 │
│  [スキップ]      [✓ 記録する]  │
└─────────────────────────────────┘
```

**設計ポイント:**
- 3秒でタグ付け完了が目標
- 動物名はインクリメンタルサーチ（animals テーブルから）
- よく使う動物を上部に表示
- スキップしても写真は保存（あとから編集可能）
