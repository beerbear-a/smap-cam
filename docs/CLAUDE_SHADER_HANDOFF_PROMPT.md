- bloom や light leak を盛って誤魔化すこと
- 「Kodak っぽい」「レトロっぽい」で逃げること

- ざらつくのにうるさくない粒状感
- 安価なレンズ付きフィルム特有の、少し甘い解像感

## このアプリの文脈

`ZootoCam` は「動物園に写ルンですを一本だけ持って行った原体験」を、カメラアプリとして再構成しているプロダクトです。

コア体験は:

- フィルムを選ぶ
- 限られた枚数で撮る
- 撮り切る
- 待つ
- 現像する
- アルバムで見返す

つまり shader の役割は、写真を派手にすることではありません。
**“あの日の記憶の温度” を写りに変換すること**です。

### 3. 写ルンです ISO800 の再現

ここは必須です。
`写ルンです ISO800` を限りなく再現した look を、逃げずに作ってください。

必要なら調整対象として、少なくとも次を検討してください。

- tone curve
- highlight rolloff
- toe / shoulder の形
- warm bias
- green / yellow の出方
- blue の落ち方
- halation
- vignette
- lens softness
- subtle chromatic aberration
- coarse / fine grain balance
- shadow noise と highlight cleanliness の差

### 4. パフォーマンス

重くしすぎるのは避けてください。
ただし「軽い代わりにフィルム感が消える」のもダメです。

必要なら:

- ライブプレビューでは軽量近似
- 現像後プレビューや静止画確認では高品質

という二層構成にしてください。
