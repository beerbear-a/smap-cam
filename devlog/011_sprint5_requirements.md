### カメラ機能 — 軸1

| ID | 機能 | 根拠 | 担当 |

### フィルムルック — 軸2（DAZZ比較）

| ID | 機能 | DAZZに対応機能あり | 担当 |
|----|------|---------|------|
| L1 | **LUT強度スライダー**（0〜100%） | ○ | Maya |
| L2 | **ライトリーク効果**（フィルム端の光漏れ） | ○ | Maya |

**LUT強度スライダーの実装方針:**
```dart
// ColorFilter.matrix の非対角成分を lerp するだけ
// identity matrix と lut matrix の間を t で補間
final matrix = List.generate(20, (i) =>
  lerpDouble(kIdentityMatrix[i], lut.colorMatrix[i], intensity)!
);
```

**ライトリークの実装方針:**
```
_GrainPainter の上に重ねる CustomPainter として実装
フィルム左端 or 右端に RadialGradient（オレンジ〜透明）を描く
強度: weak / medium / strong の3段階
```

### シェア — 軸3（重要）

| ID | 機能 | 備考 | 担当 |
|----|------|------|------|
| S1 | **Instagram Story対応（9:16書き出し）** | コンタクトシートを縦長フォーマットで出力 | Kenji |
| S2 | **透かし位置選択**（右下 / 左下 / 中央下） | Proの特典候補だが初期は全員OK | Kenji |

**9:16書き出しの実装方針:**
```
ContactSheetService に outputFormat: ContactSheetFormat enum を追加
ContactSheetFormat.square → 現行（正方形）

```
