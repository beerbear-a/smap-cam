## 自己紹介

LINEでスタンプ・カメラ機能のUXを5年、その後Mercariで「売る体験」のデザインを担当。
現在フリーランスで「感情が動くUIだけ作る」をポリシーにしている。

**信条:**
「UXは感情設計。タップするたびに何かを感じさせる。何も感じないUIは存在しないのと同じ。」
「フローに迷いが1秒でもあれば、それは設計の失敗。」

## コードベース監査結果

Kenji・Maya・Reiのコードをひと通り読んだ。
技術クオリティは高い。ただし**「体験として成立しているか」**の観点で問題が複数ある。

---

### 問題1: 最重要フロー「撮影 → 記録」の導線が見えない 🔴

**現状:**
```
起動 → MapScreen
MapScreen に FloatingActionButton（+）
→ CheckInScreen（動物園選択）
→ CameraScreen（撮影）
→ DevelopScreen（現像）
→ JournalScreen（タグ付け）
→ MapScreen（戻る）
```

**問題点:**
- 起動直後に何をすればいいかわからない
- MapScreen のFABが「チェックイン」だと気づかない（アイコンが `+` のみ）
- 「フィルムで書き出す」ボタンが3タップ奥に埋まっている（FAB → 一覧 → 詳細）

**改善案:**
```
MapScreen FAB → ラベル付き（「動物園へ」または「チェックイン」）
初回起動 → チェックインへの案内オーバーレイ（1回のみ）
```

---

### 問題2: LUTセレクターに「無料/Pro」の区別がない 🟠

**現状:** `KODAK / FUJI / MONO` が横並びで同等に見える

**問題点:**
- リリース後にProを導入したとき、ユーザーが混乱する
- 今から `FREE` バッジを設計しておかないと後でUI変更コストが大きい

**改善案:**
```
KODAK (FREE)  WARM (FREE)  FUJI ★  MONO ★
                                  ↑ ★マーク = Pro予告
```
Mayaが `isPro` フラグを追加する → Kenjiがバッジを乗せる。今日対応。

---

### 問題3: 図鑑の「空っぽ状態」が設計されていない 🟠

**現状:** encounters が0のとき、空のグリッドが表示される

**問題点:**
- 初回起動ユーザーが何もない画面を見る → 離脱
- 「まず撮りに行こう」という動機付けがない

**改善案:**
```
encounters = 0 の図鑑
→ 「まだ出会いがありません」
→ 「動物園へ行ってシャッターを切ってみよう」
→ [チェックインする] ボタン
```

---

### 問題4: 透かしプレビューが「テキスト」しかない 🟡

**現状:** settings_screen.dart の透かしプレビューは文字列のみ

**問題点:**
- ユーザーが「どんな透かしが付くか」を事前に確認できない
- シェアする前に「こうなります」を見せないと不安

**改善案:**
```
設定画面に「透かしプレビュー」ウィジェット
→ サンプル写真（グレーの矩形）に実際の透かしを合成して表示
→ WatermarkService.preview() を呼べばよい
```

---

### 問題5: BottomNavigation のタブ構造が情報量不足 🟡

**現状:** Map / 図鑑 / 設定 の3タブ

**問題点:**
- 「今撮影中のフィルム」に戻る導線がない（フィルムが現像待ちのとき迷子になる）
- シェア履歴・コンタクトシートへのアクセス経路が深い

**改善案（MVP内での対応）:**
```
Map → アクティブセッション中はバッジ表示（フィルムカウンター）
      タップ → 現在のフィルム詳細（残り枚数・動物園名）
```

## ビジュアルトーン定義

このアプリのビジュアル言語を明文化する。迷ったときはこれを参照:

```
色彩
├── 背景: #080808 〜 #181818（フィルム暗室）
├── テキスト: #FFFFFF（高コントラスト）/ #555555（サブ情報）
├── アクセント: #FFFFFF（タップ対象）/ オレンジ（警告・残り枚数少）
└── 禁止: 明るい青・緑・紫（フィルムの世界観と合わない）

タイポグラフィ
├── ヘッダー: 細字（FontWeight.w200〜w300）・レタースペース広め
├── ラベル: 大文字・letterSpacing 3〜5
└── 本文: 普通のSans-Serif、行間 1.6

インタラクション
├── タップ → Haptic（Kenjiの担当、必須）
├── 遷移 → DarkFadeRoute（暗転）、Material遷移は使わない
└── ローディング → 「LOADING」テキスト + 細いプログレス（Mayaのスタイル）
```
