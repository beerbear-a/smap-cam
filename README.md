# ZootoCam

フィルム体験を核にしたカメラアプリです。ロール単位で撮影し、現像して振り返る体験を現代的なUIで提供します。
撮影メモは自動入力され、場所のチェックイン（現在は内部データ）と結び付けて記録します。

## 主な機能

- フィルム/インスタントの撮影体験
- フィルムルック（現像後の色味）選択
- グリッド/タイマー/ライトリーク
- ロールの現像、アルバム、共有
- 図鑑タブ（出会い済み/未発見の整理）
- 撮影メモの自動入力（iOS Vision）

## 開発メモ

- Flutter (iOS/Android)
- Mapbox は現在プレースホルダー表示（タブは表示）

## 実行

```
./flutter/bin/flutter pub get
./flutter/bin/flutter run
```
